collaps – API for interfacing with COLLAPS data¶

The collaps module is designed to interface with the results from the COLLAPS setup at ISOLDE, CERN. The class MCPParser provides a universal parser for reading in the measurements made by the MCP-program. This parser outputs an instance of the class Measurement, providing a unified interface in Python to the data. If a Kepco-scan is made, the calibration fit is made automatically, and the parameters are available as attributes of the tracks in the Measurement instance.

Furthermore, several functions have been included for easier data analysis and prediction. The functions beta() and dopplerfactor() calculate the relativistic factors for the COLLAPS setup, which is needed to calculate the correct laser frequency. voltageshift(), predictspectrum() and interactiveMode() have more of a use in the prediction of spectra; their use is documented in the docstrings of the code.

satlas.collaps.voltageshift(mass, transitionFreq, laserFreq, origVoltage=30000.0, isotopeshift=0.0)[source]¶

Calculates what voltage has to be applied in order to get the resonance.

Parameters:
  • mass (float) – Mass of the isotope in amu.
  • transitionFreq (float or Energy instance) – Frequency of the transition. Can be a value (in MHz), or an instance of the Energy class.
  • laserFreq (float or Energy instance) – Frequency that the laser is tuned to. Can be a value (in MHz), or an instance of the Energy class.
Other Parameters:
 
  • origVoltage (float, optional) – Original applied voltage in volt. Defaults to 30kV.
  • isotopeshift (float or Energy instance) – Value of the isotope shift. Can be a value (in MHz), or an instance of the Energy class. Defaults to 0.
Returns:

The first value is the new value of the total applied voltage needed, the second is the shift compared to the original voltage. Note that a negative value means an increase in kinetic energy is needed. The value of shift is what the Fluke should be set to, including polarity!
Return type:

tuple
satlas.collaps.predictspectrum(mass, I, J, ABC, isotopeshift, lifetime, transitionFreq, laserFreq, fixedVoltage, kepcoFactor=50.0, channels=10000, plot=False, show=True, name=None, useMatplotLib=False)[source]¶

Based on the parameters given, return the guess for what the spectrum will look like. Warns if the needed line voltage exceeds the capabilities.

Parameters:
  • mass (float) – Mass of the isotope.
  • I (integer or half-integer) – Nuclear spin.
  • J (list of integers or half-integers) – Electronic spin, ordered as [J lower, J upper].
  • ABC (list of floats) – Values for the hyperfine constants, ordered as [A lower, A upper, B lower, B upper, C lower, C upper].
  • isotopeshift (float or Energy instance) – Isotope shift of the transition frequency, in MHz or Energy class.
  • lifetime (float) – Lifetime of the excited state, in seconds.
  • transitionFreq (float or Energy instance) – Frequency of the fine structure transition, in MHz or Energy class.
  • laserFreq (float or Energy instance) – Frequency that the laser is set to, in MHz or Energy class.
  • fixedVoltage (float) – Applied voltage in volt.
Other Parameters:
 
  • kepcoFactor (float, optional) – Kepco amplification factor, defaults to 50.0.
  • channels (int, optional) – Number of points on x-axis for evaluation, defaults to 10000.
  • plot (boolean, optional) – If True, a plot of the simulated spectrum will be shown. If the voltage excedes the limits of the possible linevoltages (meaning -10 < voltages < 10 evaluates as False somewhere), the plot is drawn in red, otherwise in black. Defaults to False.
  • show (boolean, optional) – If True, immediately show a plot of the simulated spectrum. Has no effect is plot is False. Defaults to True.
  • name (string, optional) – Isotope name. Displayed in the plot title. Defaults to None.
  • useMatplotLib (boolean, optional) – Use MatplotLib as the graphical engine, otherwise use PyQtGraph. Defaults to False.
Returns:

  • Linevoltage (NumPy array of linevoltages.)
  • Response (NumPy array of response to Linevoltage.)
satlas.collaps.fullprediction(filename)[source]¶

Parses the file given, making an estimate of how the spectrum will look like based on the parameters given. The format of the file should look like this:

[General]
TransitionFrequency : float
TransitionFrequencyUnit : string
LaserFrequency : float
LaserFrequencyUnit : string
Lifetime : float
ISOLDEVoltage : float

[IsotopeName1]
I : integer or half-integer
mass : mass in amu
Jlower : integer or half-integer
Jupper : integer or half-integer
Alower : float
Aupper : float
Blower : float
Bupper : float
Isotopeshift : float
IsotopeshiftUnit : string

For each section formatted such as [IsotopeName1], a spectrum is estimated. Also given will be the premavoltage needed for each isotope.

Parameters:filename (string) – Name of the file which contains all the data.
satlas.collaps.interactiveMode(I, J, ABC, f, filename=None, mass=None, transitionFreq=None, laserFreq=None, scalar=46, kepco=50.0)[source]¶

Given the information contained in I, J, AB and f (see the documentation of Spectrum for these parameters), a GUI is built with sliders for the most important parameters.

If the optional values filename, mass, transitionFreq and laserFreq are given, the .mcp file is loaded, converted to frequency, the laser frequency is subtracted and is displayed in the back of the spectrum. If this is so, a fit can be performed with the currently selected values.

Uses PyQtGraph to build the GUI.

Parameters:
  • I (integer or half-integer) – Nuclear spin.
  • J (list of integers or half-integers) – Electronic spin, ordered as [J lower, J upper].
  • ABC (list of floats) – Values for the hyperfine constants, ordered as [A lower, A upper, B lower, B upper, C lower, C upper].
  • f (float) – Central frequency of the spectrum.
Other Parameters:
 
  • filename (string, optional) – If this is different from None, the file is loaded and displayed in the background.
  • transitionFreq (float or Energy instance, optional) – Frequency of the fine structure transition, in MHz or Energy class. Used to convert the voltage in the file to frequency.
  • laserFreq (float or Energy instance, optional) – Frequency that the laser is set to, in MHz or Energy class. Used to convert the voltage in the file to frequency.
  • scalar (float, optional) – Select the scalar in the file. Defaults to 46.
  • kepco (float, optional) – Kepco amplification factor, defaults to 50.0.
