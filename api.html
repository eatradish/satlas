API reference¶

Spectrum creation¶

Note

The abstract baseclass Spectrum defines a few methods for retrieving information about the current state of the fit. These methods are not documented in the child classes, but will be regularly used.

satlas.spectrum.Spectrum() Abstract baseclass for all spectra, such as SingleSpectrum, CombinedSpectrum and IsomerSpectrum.
satlas.singlespectrum.SingleSpectrum(I, J, ...) Constructs a HFS spectrum, consisting of different peaks described by a certain profile.
satlas.isomerspectrum.IsomerSpectrum(spectra) Create a spectrum containing the information of multiple hyperfine structures.
satlas.combinedspectrum.CombinedSpectrum(spectra) Combines different spectra for simultaneous fitting.

Fitting routines¶

satlas.fitting Implementation of fitting routines specialised for Spectrum objects.

Lineshapes¶

satlas.profiles Implementation of classes for different lineshapes, creating callables for easy and intuitive calculations.

Utilities¶

satlas.utilities Implementation of various functions that ease the work, but do not belong in one of the other modules.
