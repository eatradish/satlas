satlas.utilities.concat_results¶

satlas.utilities.concat_results(list_of_results, index=None)[source]¶

Given a list of DataFrames, use the supplied index to concatenate the DataFrames.

Parameters:
  • list_of_results (list of pandas Dataframes) – List of DataFrames to be concatenated.
  • index (list) – List of keys to use as row-indices.
Returns:

concatenated_frames – Concatenated DataFrame
Return type:

DataFrame
