satlas.fitting.likelihood_lnprior¶

satlas.fitting.likelihood_lnprior(params)[source]¶

Calculates the logarithm of the prior given the parameter values. This is independent of the data to be fitted to.

Parameters:params (lmfit.Parameters with satlas.PriorParameters) – Collection of satlas.PriorParameters, which contain also prior bounds, so the boundary calculations are not triggered before this point. Allows easy interfacing with the emcee package.
Returns:Calculates a flat prior: returns 1.0 if all parameters are inside their boundaries, return -np.inf if one of them is not.
Return type:1.0 or -np.inf

Note

In case a custom prior distribution is required, this function has to be overwritten.
