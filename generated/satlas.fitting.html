Implementation of fitting routines specialised for Spectrum objects.

Functions

chisquare_fit(f, x, y, yerr[, xerr, func]) Use a non-linear least squares minimization (Levenberg-Marquardt) algorithm to minimize the chi-square of the fit to data x and y with errorbars yerr.
chisquare_model(params, f, x, y, yerr[, ...]) Model function for chisquare fitting routines as established in this module.
chisquare_spectroscopic_fit(f, x, y[, xerr, ...]) Use the chisquare_fit function, automatically estimating the errors on the counts by the square root.
likelihood_fit(f, x, y[, xerr, func, ...]) Fits the given model to the given data using the Maximum Likelihood Estimation technique.
likelihood_lnprior(params) Calculates the logarithm of the prior given the parameter values.
likelihood_lnprob(params, f, x, y, xerr, func) Calculates the logarithm of the probability that the data fits the model given the current parameters.
likelihood_loglikelihood(params, f, x, y, ...) Given a parameters object, a Model object, experimental data and a loglikelihood function, calculates the loglikelihood for all data points.
likelihood_walk(f, x, y[, xerr, func, ...]) Calculates the uncertainty on MLE-optimized parameter values by performing a random walk through parameter space and comparing the resulting loglikelihood values.
likelihood_x_err(f, x, y, xerr, func) Calculates the loglikelihood for a model given x and y values.
