satlas.combinedspectrum.CombinedSpectrum¶

class satlas.combinedspectrum.CombinedSpectrum(spectra)[source]¶

Combines different spectra for simultaneous fitting.

Methods

__init__(spectra) Initializes the class for simultaneous fitting of different spectra.
__call__(x) Pass the seperate frequency arrays to the subspectra, and return their response values as a list of arrays.
__add__(other) Adding another CombinedSpectrum adds the spectra therein to the list of spectra, adding an IsomerSpectrum or SingleSpectrum adds that one spectrum to the list.
plot([x, y, yerr, no_of_points, ax, show, ...]) Routine that plots the hfs, possibly on top of experimental data.
plot_spectroscopic([x, y, no_of_points, ax, ...]) Routine that plots the hfs of all the spectra, possibly on top of experimental data.
seperate_response(x) Generates the response for each subspectrum.

Attributes

params Instance of lmfit.Parameters object characterizing the shape of the HFS.
selected When a walk is performed and a triangle plot is created from the data, the parameters with one of these strings in their name will be displayed.
shared Contains all parameters which share the same value among all spectra.
