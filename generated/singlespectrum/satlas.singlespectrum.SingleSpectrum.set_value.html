satlas.singlespectrum.SingleSpectrum.set_value¶

SingleSpectrum.set_value(value, name)[source]¶

Sets the value of the selected parameter to the given value.

Parameters:
  • value (float) – Value for the parameter.
  • name (string) – Parameter name.

Note

If an iterable is supplied for both value and name, all parameters in name are set to the corresponding value.
