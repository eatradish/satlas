satlas.isomerspectrum.IsomerSpectrum.plot_spectroscopic¶

IsomerSpectrum.plot_spectroscopic(**kwargs)[source]¶

Routine that plots the hfs of all the spectra, possibly on top of experimental data. It assumes that the y data is drawn from a Poisson distribution (e.g. counting data).

Parameters:
  • x (list of arrays) – Experimental x-data. If list of Nones, a suitable region around the peaks is chosen to plot the hfs.
  • y (list of arrays) – Experimental y-data.
  • yerr (list of arrays) – Experimental errors on y.
  • no_of_points (int) – Number of points to use for the plot of the hfs.
  • ax (matplotlib axes object) – If provided, plots on this axis
  • show (Boolean) – if True, the plot will be shown at the end.
Returns:

fig, ax
Return type:

matplotlib figure and axes
