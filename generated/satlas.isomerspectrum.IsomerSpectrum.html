satlas.isomerspectrum.IsomerSpectrum¶

class satlas.isomerspectrum.IsomerSpectrum(spectra)[source]¶

Create a spectrum containing the information of multiple hyperfine structures.

Methods

__init__(spectra) Initializes the HFS by providing a list of SingleSpectrum objects.
__call__(x) Get the response for frequency x (in MHz) of the spectrum.
__add__(other) Adding an IsomerSpectrum results in a new IsomerSpectrum with the new spectrum added.
plot([x, y, yerr, no_of_points, ax, show, ...]) Routine that plots the hfs of all the spectra, possibly on top of experimental data.
plot_spectroscopic(**kwargs) Routine that plots the hfs of all the spectra, possibly on top of experimental data.
seperate_response(x[, background]) Get the response for each seperate spectrum for the values x, without background.

Attributes

params Instance of lmfit.Parameters object characterizing the shape of the HFS.
selected When a walk is performed and a triangle plot is created from the data, the parameters with one of these strings in their name will be displayed.
shared Contains all parameters which share the same value among all spectra.
