*******************************************************
:mod:`collaps` -- API for interfacing with COLLAPS data
*******************************************************

The :mod:`collaps` module is designed to interface with the results from the COLLAPS setup at ISOLDE, CERN. The class :class:`MCPParser` provides a universal parser for reading in the measurements made by the MCP-program. This parser outputs an instance of the class :class:`Measurement`, providing a unified interface in Python to the data. If a Kepco-scan is made, the calibration fit is made automatically, and the parameters are available as attributes of the tracks in the :class:`Measurement` instance.

Furthermore, several functions have been included for easier data analysis and prediction. The functions :func:`beta` and :func:`dopplerfactor` calculate the relativistic factors for the COLLAPS setup, which is needed to calculate the correct laser frequency. :func:`voltageshift`, :func:`predictspectrum` and :func:`interactiveMode` have more of a use in the prediction of spectra; their use is documented in the docstrings of the code.
